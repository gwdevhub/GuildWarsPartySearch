# GuildWarsPartySearch
Party Search Aggregator

## Build & Launch
Check the README files in each active subproject:
- `GuildWarsPartySearch.NodeJSServer` - Node.js server and web client
- `GuildWarsPartySearch.Bot` - C++ bot

## Usage
- Launch `GuildWarsPartySearch.NodeJSServer`
- Build and run a bot from `GuildWarsPartySearch.Bot`
- Go to http://localhost
